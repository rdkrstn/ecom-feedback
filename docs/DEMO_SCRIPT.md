# Demo Script

## 1. Problem

Ecommerce teams get valuable feedback from support tickets, reviews, returns, and comments, but that feedback usually stays trapped in support.

## 2. Submit sample ecommerce feedback

Open the web app, load the compression leggings sample, and create a review.

## 3. Show review created

Point out the review ID, draft status, and copyable Band kickoff message.

## 4. Show Band kickoff message

Explain that real Band mode starts when the kickoff message is pasted into a manually created Band room with all six remote agents added.

## 5. Show agents posting structured handoffs

Run local mock workflow for the demo. The timeline shows Intake Review, Customer Feedback Review, UGC Review, Support Review, Product Page Review, and Action Review posting structured outputs and handoff targets.

## 6. Show final action plan

Show the final plan: primary customer issue, UGC briefs, support macros, FAQ updates, product page tasks, owner handoff, and source evidence.

## 7. Explain business value

The workflow turns repeated customer questions into practical work for Content, Support, Storefront, and Operations. It reduces repeated support questions, improves buyer confidence, and makes feedback traceable through Band.
